<h1>PNGeez</h1>
Simple, no-nonsense image viewer. Written in Python with Pygame.
<h1>Building</h1>
<h2>Windows</h2>
<b>Prerequesites:</b><br>
<ul>
    <li>Python installed</li>
    <li>Pygame installed</li>
    <li>Pyinstaller</li>
</ul>
To build, run <code>build.bat</code>. This will populate the <code>pngeez</code> directory.
Once compiling has succeeded, copy the directory to <code>C:\Program Files\</code>, and for extra points you
can add a shortcut in <code>C:\ProgramData\Microsoft\Windows\Start Menu\Programs</code>.
